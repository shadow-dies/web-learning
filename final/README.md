# 项目运行说明

## 启动客户端

```bash
final文件打开终端

cd fe
#进入前端项目文件夹
npm i
# 安装依赖
npm start
# 运行项目
在 http://localhost:3000/ 打开前端项目
```

## 启动服务端
```bash
cd server

npm i

npm start

在3001端口启动服务器
```
